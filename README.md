# COVID-19 Dashboard

A comprehensive COVID-19 data analysis and visualization project built to monitor the spread of COVID-19 across different regions. The dashboard provides insights into confirmed cases, active cases, recoveries, deaths, and vaccination trends using interactive charts and maps.

## Features
- Global and country-wise COVID-19 statistics
- Confirmed, recovered, active, and death cases
- Vaccination progress tracking
- Interactive dashboards and visualizations
- Trend analysis over time
- Data filtering by country and date

## Technologies Used
- Power BI / Tableau
- SQL
- Python (Pandas, NumPy)
- Excel
- CSV Datasets

## Data Source
- Johns Hopkins University COVID-19 Dataset
- Our World in Data
- World Health Organization (WHO)

## Purpose
This project helps users understand COVID-19 trends through data visualization and supports data-driven decision-making.# covid-19-dashboard
